## 1. 业务术语

### 1.1 核心概念

| 术语 | 英文 | 定义 | 示例 |
|------|------|------|------|
| 备课 | Lesson Planning | 教师为授课准备的教学方案和内容 | 为初二数学课准备一次函数的教案 |
| 教案 | Lesson Plan | 详细的教学设计方案，包含教学目标、内容、方法等 | 《一次函数的概念》教案 |
| 学情 | Student Profile | 学生的学习情况，包括基础、习惯、特点等 | 张三数学基础薄弱，学习习惯良好 |
| 知识点 | Knowledge Point | 学科中的具体知识单元 | 一次函数的定义、图像、性质 |
| 习题 | Exercise | 用于练习和巩固知识的题目 | 一次函数练习题 |
| 教学目标 | Teaching Objective | 课程要达到的学习成果 | 理解一次函数的概念 |
| 教学重难点 | Key & Difficult Points | 教学中的重点和难点内容 | 重点：一次函数概念；难点：实际应用 |
| 学生画像 | Student Persona | 基于学生信息构建的立体化描述 | 初二、数学薄弱、性格内向 |

### 1.2 用户角色

| 术语 | 英文 | 定义 | 说明 |
|------|------|------|------|
| 家教老师 | Tutor | 提供一对一或小班教学服务的教师 | 平台主要用户群体 |
| 专职家教 | Full-time Tutor | 以家教为主要职业的教师 | 学生数量多，对效率要求高 |
| 兼职家教 | Part-time Tutor | 利用业余时间做家教的人员 | 如大学生、在职教师 |
| 机构教师 | Institution Teacher | 培训机构的全职教师 | 学生数量多，需要满足机构要求 |
| 家长用户 | Parent User | 使用平台辅导孩子的家长 | 高学历，自助教学 |

### 1.3 功能模块

| 术语 | 英文 | 定义 | 说明 |
|------|------|------|------|
| 智能备课 | AI Lesson Planning | 利用AI技术自动生成备课内容 | 平台核心功能 |
| 学生管理 | Student Management | 管理学生档案和学情信息 | 基础功能模块 |
| 教学资源 | Teaching Resources | 知识点库、题型库、模板库等 | 辅助教学资源 |
| 教学进度 | Teaching Progress | 跟踪和管理教学进度 | 进度可视化 |
| 学习报告 | Learning Report | 学生学习情况的总结报告 | 可分享给家长 |

## 3. 备课专业术语

### 3.1 备课模式

| 术语 | 定义 | 适用场景 | 生成重点 |
|------|------|----------|----------|
| 新课讲解 | 首次学习新知识点 | 学生初次接触某个概念 | 概念讲解、基础例题 |
| 习题讲解 | 讲解练习题 | 巩固已学知识 | 错题分析、举一反三 |
| 考前复习 | 考试前集中复习 | 期中/期末考试前 | 知识点串讲、易错点 |
| 查漏补缺 | 针对薄弱环节强化 | 发现知识盲区后 | 专项练习、强化训练 |
| 培优拔高 | 提升优秀学生 | 基础好的学生 | 拓展知识、难题训练 |
| 假期预习 | 提前学习下学期内容 | 寒暑假期间 | 导学、基础掌握 |

### 3.2 教案结构

| 术语 | 定义 | 内容说明 |
|------|------|----------|
| 教学目标 | 课程要达到的学习成果 | 知识目标、能力目标、情感目标 |
| 教学重点 | 需要重点掌握的内容 | 核心概念、关键技能 |
| 教学难点 | 学生较难理解的内容 | 抽象概念、复杂应用 |
| 教学过程 | 教学活动的详细安排 | 导入、讲解、练习、总结 |
| 时间分配 | 各环节的时间安排 | 导入5分钟、讲解30分钟等 |
| 教学方法 | 采用的教学策略 | 讲授法、讨论法、演示法等 |
| 教学手段 | 使用的教学工具 | 多媒体、教具、实验等 |
| 板书设计 | 板书内容的规划 | 主板书、副板书 |
| 课后作业 | 课后练习内容 | 巩固练习、预习任务 |
| 教学反思 | 课后总结和改进 | 效果评估、改进建议 |

### 3.3 题型分类

| 术语 | 定义 | 特点 |
|------|------|------|
| 选择题 | 从选项中选择正确答案 | 考查识别能力 |
| 填空题 | 填写正确答案 | 考查记忆能力 |
| 判断题 | 判断正误 | 考查理解能力 |
| 解答题 | 详细解答过程 | 考查综合能力 |
| 证明题 | 数学证明过程 | 考查逻辑推理 |
| 应用题 | 实际问题解决 | 考查应用能力 |
| 探究题 | 开放性探索 | 考查创新能力 |

## 5. 产品文案规范

### 5.1 通用文案

| 场景 | 推荐文案 | 避免使用 |
|------|----------|----------|
| 欢迎语 | 👋 早上好，张老师！ | 您好，用户 |
| 操作成功 | ✓ 保存成功 | 操作已完成 |
| 操作失败 | ✗ 保存失败，请重试 | 错误 |
| 加载中 | ⏳ 正在生成备课内容... | 加载中... |
| 确认删除 | 确认删除？删除后无法恢复 | 确定删除？ |
| 空状态 | 还没有学生，添加第一个学生吧 | 暂无数据 |

### 5.2 功能名称

| 功能 | 产品内名称 | 说明 |
|------|------------|------|
| AI备课生成 | 智能备课 | 简洁易懂 |

| 学情信息 | 学习情况 | 通俗易懂 |
| 备课历史 | 我的备课 | 个人化 |
| 知识点库 | 知识库 | 简洁 |
| 学习报告 | 学情报告 | 专业说法 |

### 5.3 错误提示

| 错误类型 | 提示文案 | 建议操作 |
