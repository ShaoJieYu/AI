## 📊 项目概览

### 产品定位
一个面向家教老师的智能备课平台，通过 AI 技术实现个性化备课内容自动生成，让备课效率提升 10 倍。

### 核心功能
1. **学生信息管理**: 完整的学生档案和学情记录
2. **AI 备课生成**: 基于学生情况自动生成个性化教案
3. **教案编辑优化**: 支持在线编辑和版本管理
4. **习题生成**: 智能生成针对性练习题
5. **多端协同**: Web、App、小程序三端数据实时同步

### 目标用户
- 专职家教老师
- 兼职大学生家教
- 培训机构教师
- 家长自助教学

### 技术栈
- **前端**: React 18 + TypeScript + Flutter
- **后端**: Spring Boot 3.2 + Java 21
- **AI 服务**: Python + FastAPI + 国产大模型
- **基础设施**: MySQL + Redis + MinIO
